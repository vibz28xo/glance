## 📋 What is glance?

glance is a simple and modern web tool to manage Samba (SMB) servers on your local network or NAS device. It offers a clean interface that lets you handle files, stream media in real time, and manage SSH keys without fuss. glance saves bandwidth while uploading files and supports live streaming of content stored on your Samba shares.

You do not need technical skills to use glance. The interface is designed for everyday users who want access to their network files with ease from a web browser.

---

## 🔍 Main Features

- **File manager:** Browse, upload, download, delete, and move files on your Samba server.
- **Real-time file streaming:** Watch videos or listen to audio files directly from your SMB shares.
- **Bandwidth-saving uploads:** Upload only changed parts of large files to save data.
- **SSH key management:** Manage SSH keys used for secure server access.
- **Clean web interface:** Uses a smooth glassmorphism style for a modern look.
- **Self-hosted:** All your data stays on your network; no third-party services.
- **Works on Windows:** Runs easily on Windows computers with no programming needed.

## ⚙️ System Requirements

- **Operating System:** Windows 10 or later  
- **Processor:** 2 GHz or faster

## 🛠️ Troubleshooting

If you face issues when running glance on Windows, try these steps:

- Check that your Windows firewall allows glance to communicate on your network.
- Make sure your Samba server is running and accessible from your Windows device.
- Confirm you entered the correct Samba username and password.
- Restart glance and your computer if needed.
- Use a different web browser if the interface does not load.

For detailed logs, look inside the installation folder for log files.

## 🧰 How glance works behind the scenes

glance runs as a local server on your Windows PC. It communicates with your Samba server using SMB protocols to list and manage files. The web interface connects to this local server in your browser. This design keeps your data secure and stored within your own network.
